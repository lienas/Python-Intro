US Bikeshare Data Analysis

This Python program analyzes bikeshare data from three major US cities:
Chicago, New York City, and Washington.

Features:
- Filter data by city, month, and day of week
- Display most common travel times (month, day, hour)
- Show most popular stations and trip combinations
- Calculate total and average trip duration
- Provide user statistics (user types, gender, birth year)

The program presents trip durations in a readable format (days, hours,
minutes, seconds) and allows users to interactively explore the data.

Files:
- bikeshare_2.py: Main program (Flake8 compliant)
- chicago.csv, new_york_city.csv, washington.csv: Data files (not provided)
